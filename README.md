# Orbit Striker (Phaser 3 Edition)

論理的なシステム設計に基づき、外部アセットを一切使用せずコードのみで構築されたブラウザ・シューティングゲーム。

## 🛠 設計のコア・コンセプト

### 1. アセットレス・アーキテクチャ
外部画像ファイル（PNG/JPG）を読み込むのではなく、Phaserの `Graphics` クラスを使用して、実行時にすべてのテクスチャをプログラムで描画します。これにより、以下のメリットを確保しています。
- **高速なロード**: HTTPリクエストを最小限に抑え、瞬時にゲームを開始。
- **スケーラビリティ**: コード上の数値を変更するだけで、デザインやサイズを即座に変更可能。

### 2. 衝突判定の幾何学的最適化
標準の矩形（四角形）判定ではなく、物体の形状に合わせた円形当たり判定（Circular Collision）を実装。
- 岩（Rock）の描画半径と判定半径を完全に同期。
- プレイヤーの当たり判定を胴体部分に絞り込むことで、プレイの快適性を向上。

### 3. リソース・マネジメント
`update` ループ内でのライフサイクル管理を徹底。
- 画面外に消失した弾丸および岩石オブジェクトを即座にメモリから解放。
- 長時間実行時のメモリリークを防止。

## 🎮 操作方法
- **左右矢印キー**: 自機の移動
- **スペースキー**: レーザー発射
- **RESTARTボタン**: ゲーム再開

## 🚀 技術スタック
- **Framework**: Phaser 3.60.0
- **Physics**: Arcade Physics
- **Language**: JavaScript (Vanilla)
